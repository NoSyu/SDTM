# Self-Disclosure Topic Model
Implementation of [Self-Disclosure Topic Model (SDTM)](https://www.aclweb.org/anthology/D14-1213/) in EMNLP 2014.

## Environment
We run this code in this environment.

- Hardware
    - Intel CPU 16 cores
    - 128GB memory

- Software
	- Microsoft Windows
    - Visual Studio 2012
